Para executar o programa:
	python3 main.py [modulos]

	- OPÇÕES:
		Modulos:
			* gli: gerador de lista invertida
			* pc: processador de consultas
			* index: indexador
			* busca: buscador

Exemplos:
	* Executar apenas o módulo gli:
		python3 main.py gli
	* Executar todos os módulos:
		python3 main.py


Descrição das pastas:
	cfg: arquivos de configuração
	data: arquivos da base base CysticFibrosis2
	modules: arquivos fontes dos módulos
	output: diretório de saída de arquivos gerados pelo sistema
